How to add your signature

Gmail's built-in signature editor strips raw HTML — animations and advanced formatting won't survive a direct paste. Use one of the methods below instead.

Recommended — WiseStamp Extension

  1. Install WiseStamp from the Chrome Web Store (free tier available)
  2. Open WiseStamp → click Create New Signature
  3. Switch to the HTML editor tab inside WiseStamp
  4. Click Copy Signature HTML above and paste it in
  5. Save — WiseStamp automatically injects it into Gmail on every compose

Alternative — Basic paste (formatting only, no animation)

  1. Open Gmail → Settings ⚙ → See all settings
  2. Scroll to Signature → click Create new
  3. Click Copy Signature HTML above, then paste directly into the signature box — Gmail will preserve basic formatting but strip CSS animations
  4. Scroll down and click Save Changes

💡 Other options that support animated/HTML signatures: SignaIA, Signaware (GIF-based), and Signature Studio. All inject signatures via a Chrome extension.

  1. Click Copy Signature HTML button above
  2. Open Outlook → File → Options → Mail → Signatures…
  3. Click New, name your signature, then click OK
  4. In the signature editor click the Source / HTML view button
  5. Paste the copied HTML and switch back to normal view
  6. Set it as default for New messages and click OK

💡 Tip: On Outlook for Mac, go to Preferences → Signatures to find the same option.

  1. Click Copy Signature HTML button above
  2. Open Mail app → Mail menu → Settings → Signatures tab
  3. Click + to add a new signature and name it
  4. Uncheck "Always match my default message font"
  5. Open TextEdit, paste the HTML, save as .html file
  6. Drag the saved .html file into the Apple Mail signature text area

💡 Apple Mail doesn't have a direct HTML paste option — dragging the saved file is the most reliable method.
